## Walking-Diary 보행자 걸음걸이 분석 프로젝트
### 무엇을 만드는가

평소 많이 걷거나, 자신의 걸음걸이를 교정하고 싶은 사람들을 대상으로 머신러닝을 이용해 자이로 센서로 측정, 저장한 걸음걸이 데이터를 분석한다. 이 데이터를 통해 사용자에게 자신의 걸음 습관에 대한 정보를 제공하고 팔자걸음과 같은 잘못된 걸음 습관을 인식하고 교정할 수 있도록 하는 IOT 서비스를 개발한다.
